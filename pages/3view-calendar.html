`~#$3VIEW CALENDAR

 

DESCRIPTION: This page displays the Canny Communicator Calendar to the user. The user must be logged in to the system to view the calendar. The user can view the calendar with all the events. The calendar personalised for each user as events differ per user. This page also displays the Notices and messages for the user to view.
