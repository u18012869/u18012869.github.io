`~#$3 UPDATE PROGRESS REPORTS

 

DESCRIPTION:  This page allows the user (teacher) to update a student’s Progress Report. The user must be logged into the system to update progress reports. To update a progress report the user is required to update the following details: Date, Pupil Name and Surname, Pupils Age, Weight and Hight, Social and emotional development details, Body awareness details, Fine motor coordination details, Visual perception details, Language details. The Progress report details will be updated and saved for the student when the Page is completed, and the user clicks the Save Progress Report button.
