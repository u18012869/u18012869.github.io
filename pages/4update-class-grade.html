`~#$4UPDATE CLASS GRADE

 

DESCRIPTION:

This popup is used to update class grade on the system. The popup is populated with the selected class grade information. The user can enter a new class grade name. The update button should then be selected to submit the form and update the class grade.

Allows the user to close the popup screen without updating new information
