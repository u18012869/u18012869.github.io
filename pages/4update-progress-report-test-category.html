`~#$4UPDATE PROGRESS REPORT TEST CATEGORY

 

DESCRIPTION: This popup is used to update the progress report test category on the system. The popup is populated with the selected progress report test category information. The user can enter a new progress report test category name. The update button should then be selected to submit the form and update the progress report test category.
