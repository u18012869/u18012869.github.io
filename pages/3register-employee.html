`~#$3REGISTER EMPLOYEE

 


DESCRIPTION:  This page allows the admin to add a new teacher to the system. The admin will enter the Name, Surname, Contact Number and Email Address  of the new teacher. The admin will also assign the new teacher to a class here by selecting one of the classes in the Class Drop Down. The admin can then submit the new information by clicking the Add Button, which will save the information. If the admin wishes to cancel this request, he/she can click the cancel button . The same description applies for ADDING A NEW SUPPORT STAFF MEMBER, however instead of being assigned to a class the employee can be assigned to a role.
