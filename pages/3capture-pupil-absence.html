`~#$3CAPTURE PUPIL ABSENCE

 

DESCRIPTION:

 This page allows the user (teacher) to capture a pupil’s absence i.e. mark a student absent. The user must be logged into the system to capture pupil absence. To capture the pupil absence the user is required to select a class. The class list for that class will then be displayed. The user must then mark each pupil either present or absent and give a reason for absence. Pupil absence will be captured and saved for the student when the Page is completed.
