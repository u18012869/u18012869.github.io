`~#$3UPDATE EXTRA-MURAL

 

DESCRIPTION: This popup is used to update extra murals on the system. The popup is populated with the selected extra mural information. The user can enter a new day, extra mural name, extra mural description, and the cost. The update button should then be selected to submit the Page and update the extra mural.
