`~#$3VIEW MEAL PLAN

 

DESCRIPTION:  This page allows the admin to view the most current meal plan for the week. The meal plans are separated by each day of the week. The meal plans type, and food is displayed. From there the admin can Add, Update or Delete a meal plan by clicking the corresponding buttons.

 

 

NUMBER

ITEM NAME

DESCRIPTION

1

Add information Label

Used to indicate how to add a new meal plan

2

Add Button

Clicking this will open up the ADD MEAL PLAN MODAL

3

Meal Plan Table

Used to display the different meal plans under each day.

4

Update Button

Clicking this will open the  UPDATE MEAL PLAN MODAL

5

Delete Button

Clicking this will permanently delete the selected meal plan.
