`~#$3VIEW WAITING LIST

 

DESCRIPTION: This page allows the admin to view all the applications that are currently on the waiting list by clicking the Waiting List button. The applications will be separated into grade. The admin will be able to view all the details of each application by clicking the details button. The admin can then decide to accept or reject the application.
