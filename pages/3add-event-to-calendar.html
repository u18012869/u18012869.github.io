`~#$3ADD EVENT TO CALENDAR

 

DESCRIPTION: This page allows a user to add an event to the calendar on the Canny Communicator website. The user must be logged in to the system to add an event. To add an event the user is required to enter the following details: Event Name, Event Description, Event Cost, Event Date, Event Type and to upload any Event Documents needed. the Event will be added to the calendar when the Page is completed, and the user clicks the Submit button.
