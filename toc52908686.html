DESCRIPTION: This page allows the admin to view all the family accounts that are currently enrolled in the school. The user will be able to filter these results by Child Name, Surname or Class by using the search functionality. From here admin will be able to view all the details of the specified account, update these details or delete the entire family account by clicking the corresponding buttons.
