`~#$4VIEW PROGRESS REPORT TEST CATEGORY

 

DESCRIPTION: This page allows the admin to view all the progress report test categories and assessment criteria grouped by their category that is on the system. On this the admin can add new progress report test categories and assessment criteria to the system, initiate the update functions for assessment criteria and delete progress report test category and assessment criteria from the system.
